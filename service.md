## <span id="TOS">预约须知</span>

### 一、服务简介

1. 服务范围：各类笔记本、台式机的软硬件问题。由于台式机问题多半需要上门，因此处理速度可能低于笔记本的处理速度。

2. 服务原则：<text style="color: rgb(255, 255, 97); font-weight: bold;">先预约，再处理。</text>IT 侠秉承自愿原则的后台处理模式：

   - 首先，求助者在预约系统提交预约。**由于人手有限，原则上未预约者自行前往值班将不予受理。** 已经预约过的求助者用同样的手机号码在预约界面可以直接登录，并可查看、修改、删除当前自己的预约。能够看见当前预约记录即为预约提交成功。
   - IT 侠成员有空闲时会登陆后台查看所有预约的情况，选择自己能力内或者愿意维修的预约。
   - 之后，IT 侠成员会**主动联系**求助者约定处理时间。
   - 一般情况下，在您提交预约一周之内，IT 侠会与您短信（或其他方式）确认并安排处理时间，为您提供服务。如果您需要联系 IT 侠以询问预约进度，请到本页底部寻找<a href="#groups">您所在校区的在线互助群</a>询问。

3. 值班时间：IT 侠志愿者均为在校学生，时间精力有限，只能在晚上安排值班。两个校区由于条件差异，值班日期略有不同。鼓楼因为场地的特殊情况，请以与你联系的 IT 侠的通知为准。

   - 仙林：每周工作日 10:00 – 22:00
   - 鼓楼：（一般情况）每周周二、周六 18:30 – 21:30，以联络时通知为准，每学期会动态调整。
   - 苏州：周六晚 7:30 后

4. 值班地点
   - 仙林：大学生活动中心 412
   - 鼓楼：（一般情况）物理楼 233 或物理楼 227，以联络时通知为准
   - 苏州：仁园甲 103

### 二、服务条款

【此条款更新于 2023年3月1日】

IT 侠致力于为南京大学师生提供免费的计算机软硬件维修服务，以及普及计算机基础知识。在委托方接受我们的维修服务时，适用此条款。请在充分阅读并理解此条款的基础上，预约并接受 IT 侠的维修服务。
### 绪论与先决条件

1. 原则上，欲接受 IT 侠的维修服务，必须先在 IT 侠预约系统填写预约单，并在社员预先联系后再前往工作室进行维修。对于十分紧急的情况（如进水等），可前往工作室再填写预约单，但 IT 侠不保证此情况下有成员接单。
2. IT 侠的服务对象仅限南大在校师生和校友，以及其他为 IT 侠提供过捐助的爱心人士。
3. IT 侠的服务不与任何商业团体挂钩。
4. IT 侠将视用户的个人资料为绝密信息，保护隐私不外泄。
5. IT 侠人员、能力有限，在处理预约单时会适当考虑优先级，且不保证绝对的接单率。如较为紧急，需要得到尽快处理，请及时通过相关群聊、QQ 频道联系我们。
6. IT 侠是纯公益性组织，提供的服务 **不收取任何费用** ，仅在需要购买零件等情况会产生费用。任何的物料购买均会与委托者完全透明地协商（具体定义见后文）后进行。任何替换下的零件均归委托方所有，可自行带回；
7. 服务提供最终解释权归 IT 侠所有。
8. 此服务条款自2023年3月1日开始实施，并对此时间后产生的预约单有效。

### 不提供的服务

9. IT 侠不提供部分具有法律或规章风险、争议的服务：
   1. 盗版软件安装：如有安装需求，只能帮助安装试用版。对于安装复杂的专业软件，可以一起探讨，但并不能保证完全解决问题。
   2. 盗版系统安装：按照我们的维修经验，同时为了避免各类非纯净系统、激活软件带来的诸多使用问题，IT 侠只会安装正版操作系统。值得一提的是，国内大部分行货电脑均预装正版 Windows 且重装仍可用，同时校内人员亦可使用学校购买的正版系统，因此装系统基本不会产生额外费用。
   3. 科学上网/翻墙：只能帮助解决因此造成的网络问题，但不会帮助配置正常使用该类工具。
   4. 课程作业相关：原则上，IT 侠不处理课程相关的软件与环境配置问题，相关问题应当寻求对应课程的助教的帮助。同时，IT 侠不帮助任何课程作业、考试问题。

### 软硬件与数据免责事项

10. 依照行业惯例，当维修遇到以下情形时，委托方需要自行承担相应责任：
   1. 若委托方将设备或设备的一部分寄存在IT侠，而本人不在场时，IT侠将尽可能保证设备及内部数据安全，但 **不承担保管责任** ，即丢失概不负责；
   2. 设备在退（换）期内，仍要维修者，委托方需要承担因维修而无法退（换）货的风险；
   3. 设备在生产方或任何第三方的免费售后（保修）期限内，仍要维修者，委托方需要承担因 **硬件** 维修而 **失去免费保修的风险** （通常软件维修不会导致掉保）；
   4. 设备存在进水、短路、静电击穿、电池异常（如鼓包、破损等）、摔落等情况，已经被修理过，或购买自二手市场，仍要维修者，委托方需要承担维修过程中可能出现的任何风险与一切后果；
   5. 设备由于硬件设计，部分维修任务可能十分困难（如 Surface 系列的所有硬件硬件维修，或使用液态金属导热的设备进行清灰）。我们会在接单前提前告知委托方此情况。仍希望进行相应维修任务的，委托方需要承担设备损坏的风险与后果；
   6. 设备更换上了 **委托方自行购买的配件** ，委托方需要承担因为此配件而造成机器损坏的风险与后果；
   7. 设备为处于不稳定边缘、或有潜在故障风险的机器（如显卡门机器、挖矿机），仍要维修者，委托方需要承担维修过程中可能出现的任何风险与一切后果；
   8. 由于设备老化、摔落或用料低端，在拆装过程中若可能发生的卡扣折断、螺丝座碎裂等情况，IT 侠不承担责任；
11. 依照行业惯例，对于委托方设备内的数据：
   1. 在所有操作开始前，及关键步骤开始前，IT 侠会告知委托方可能的数据丢失与损坏；
   2. 建议委托方在前往 IT 侠之前预先将 **数据备份到外部设备或云端**。IT 侠亦可帮助委托者备份数据到其自己的设备或 IT 侠的设备；
   3. 对于大部分需要拆机的硬件维修，IT 侠会选择取下机器的硬盘，以最大限度防止数据丢失。当维修跨越多天时，委托方需要自行保管硬盘，我们会提供必要的包装、保管材料；
   4. 对于备份到 IT 侠设备的数据，我们将把数据保存至预约单完成，后从系统中彻底删除。整个期间如无特别必要且征得委托方同意，IT 侠任何人不会查看备份的内容；
   5. 在 IT 侠充分告知数据安全风险且提供备份服务后，因为委托方疏忽（如忘记备份某内容、不正确保管硬盘致损坏）而造成 **数据丢失的，由委托方自行承担后果** ；
   6. 在 IT 侠维修过程中，我们将会做到“非礼勿视”，在涉及潜在 **隐私风险** 的操作（如打开个人文件夹以进行备份）前，会与您确认并取得许可后再进行；

### 二次损坏责任问题

12. 依照行业惯例，在维修后造成设备损坏（或比维修前损坏情况更严重）的，按以下情况处理：
   1. **对于任何情况，我们不保证维修的时效性。我们将尽快将设备维修好，且会提供备用的笔记本电脑供应急。** 但如果遇到委托方需要离校，或寒暑假将要开始、IT 侠将暂停活动等情形，同时自己又急用电脑或完全无法接受短期内无法使用自己的设备（特别是无法接受使用备用机的），请谨慎预约。
   2. 如果 IT 侠操作上无过失（如进水机在充分干燥处理后通电仍无法使用、换上委托者自行购买的配件后无法开机），或属于上述任何一种 IT 侠免责情况，则 IT 侠不承担责任，但可以提供寻求第一方或第三方售后服务的协助与指导；
   3. 如果 IT 侠在工作上存在过失（误操作），但委托的设备亦存在问题（如曾摔落、有委托方没有描述的暗病等），则原则上，IT 侠与委托方承担的费用比例为 1:3，IT 侠可以提供寻求第一方或第三方售后服务的协助与指导；
   4. 如果 IT 侠在工作上存在过失，委托设备无问题，且不属于上述任何一种 IT 侠免责情形，则原则上，IT 侠与委托方承担的费用比例为 1:1，且 IT 侠将尽力提供寻求第一方或第三方售后服务的协助与指导；
   5. 在合理范围内，上述费用比例可据实际情况进行浮动；
   6. 关于产生的费用，进行如下的界定：
      1. 对于需要购买其它物料维修的，IT 侠将与委托人在完全透明的情况下协商购买，并以最后购买产生的费用为准；
      2. “完全透明的情况下协商”（上下同）具体指，使用可留档的通讯工具（E-Mail等保存EML文件，如果为QQ则需要存留所有聊天截图），在IT侠方仅提供零件型号、搜索关键字、可能的平台等后，由委托方寻找相关的购买或维修方式购买。
      同时，委托方可以委托IT侠代为购买、维修，分为全权购买，或IT侠代找到相关链接、地址后，委托方购买/前往维修。这两种委托方式均需购买方通过上述的通讯工具提供购买与支付证明给另一方用于价格界定。如果使用任意形式的委托购买，表明委托方同意由IT侠选定购买来源、维修站，并认可IT侠将无偏向、无利益输送地选择购买来源、维修站。
      3. 对于需要专业服务进一步维修的，IT 侠将与委托人在完全透明的情况下进行线上询价。线上询价失败的，将由 IT 侠成员赴第一方或第三方维修点线下询价，此过程将对委托方透明（通过一同前往等形式）。分摊费用以最终产生的费用为准；
      4. 如果委托方因自身因素无法接受线上或线下询价结果，或无法接受由 IT 侠成员对接第一或第三方维修服务（如急用且需要离校）的，IT 侠将一次性赔偿线上或线下询价的最低值（注意：如果委托方完全拒绝进一步的检视以定下最终价格，则分摊费用将可能为第一/三方开出的相当低的“检查费”，而非最终费用！）的 50%，并不对后续的维修承担责任；
